# Agentic Security Auditor
## Google Cloud Rapid Agent Hackathon 2026

An AI-powered smart contract security auditing agent built with Google ADK (Agent Development Kit).

### What It Does
- Analyzes smart contract code for security vulnerabilities
- Generates structured audit reports with severity ratings
- Provides remediation recommendations
- Supports multiple audit frameworks (Slither, Mythril integration)

### Built With
- Google ADK (Agent Development Kit)
- Vertex AI Agent Builder
- Gemini API
- Python 3.10+

### Quick Start
```bash
pip install -r requirements.txt
python src/agent.py
```
